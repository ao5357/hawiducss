Hawidu CSS

The CSS framework you should be using.

del/ins diff

Makes <del> and <ins> look like a diffs. Each diff-classed container on the page is incremented with a counter. To use, augment the body, then add the .diff stuff to the utility classes.

body{counter-reset:diff;} .diff{font-family:Consolas,"Bitstream Vera Sans Mono","Courier New",monospace;} .diff:before{content:"@@ diff " counter(diff);counter-increment:diff;display:block;} .diff>del,.diff>ins{display:block;padding-left:3.5ex;position:relative;text-decoration:none;white-space:pre;white-space:pre-wrap;word-wrap:break-word;} .diff>del{background:#FCC;color:#700;} .diff>del:before{content:"-";left:0;position:absolute;} .diff>ins{background:#CFC;color:#070;} .diff>ins:before{content:"+";left:0;position:absolute;}

Example

For markup that looks like this:

<div class="diff"> <del>Brad is 25 years old</del> <ins>Brad is a 26-year-old</ins> </div>
Brad is 25 years old Brad is a 26-year-old
