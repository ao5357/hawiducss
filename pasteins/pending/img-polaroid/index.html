Polaroid Images

Styles applied entirely to images to make them look like Polaroids. Serious question: Is .polaroid a semantic class name? To use, just paste the following into your styles.css, then class images you want to look like polaroids with .polaroid. Siblings will behave like a stack.

.polaroid{border:.1em solid #ccc;box-shadow:0 0 0 1em #fff,0 1em 0 1em #fff,1em 3em 1em 0 rgba(0,0,0,.5);margin:0 2em 5em;} .polaroid:nth-of-type(3n+1){-moz-transform:rotate(-3deg);-ms-transform:rotate(-3deg);-o-transform:rotate(-3deg);-webkit-transform:rotate(-3deg);transform:rotate(-3deg);} .polaroid:nth-of-type(3n+2){-moz-transform:rotate(3deg);-ms-transform:rotate(3deg);-o-transform:rotate(3deg);-webkit-transform:rotate(3deg);transform:rotate(3deg);}
